## 三、认证授权

### 3.1 认证方案

**认证方式**：

| 认证方式 | 场景 | 安全级别 |
|----------|------|----------|
| 密码认证 | 普通登录 | 中 |
| 短信验证码 | 快速登录 | 中 |
| 扫码登录 | 安全登录 | 高 |
| 人脸识别 | 高安全场景 | 高 |

**密码安全**：

| 安全措施 | 实现方式 |
|----------|----------|
| 密码强度 | 8位以上，包含大小写字母、数字、特殊字符 |
| 密码加密 | BCrypt加密存储 |
| 密码防破解 | 登录失败5次锁定账号30分钟 |
| 密码传输 | HTTPS加密传输 |

**Token设计**：

```json
{
  "header": {

    "typ": "JWT"

  "payload": {

    "name": "用户名",
    "roles": ["role1", "role2"],
    "permissions": ["perm1", "perm2"],
```

### 3.2 授权方案

**RBAC模型**：

```
用户(User) → 角色(Role) → 权限(Permission) → 资源(Resource)
```

**权限设计**：

| 角色 | 权限范围 |
|------|----------|
| 用户 | 查看自己的订单、个人信息 |
| 商家 | 管理自己的商品、订单 |
| 客服 | 查看所有订单、处理售后 |
| 管理员 | 系统管理、用户管理 |

**权限控制**：

```java
@PreAuthorize("hasPermission('order', 'read')")
public Result<Order> getOrder(Long id) {
    // 业务逻辑
}

@PreAuthorize("hasRole('admin')")

    // 业务逻辑
}
```

## 四、数据安全

### 4.1 数据加密

**加密策略**：

| 数据类型 | 加密方式 | 说明 |
|----------|----------|------|
| 密码 | BCrypt | 单向加密，不可逆 |
| 身份证 | AES | 可逆加密，脱敏展示 |
| 手机号 | AES | 可逆加密，脱敏展示 |
| 银行卡 | AES | 可逆加密，脱敏展示 |
| 支付密钥 | RSA | 非对称加密 |

**加密实现**：

```java
// AES加密
public String encrypt(String plainText, String key) {
    AES aes = new AES(Mode.CBC, Padding.PKCS5Padding,
        new SecretKeySpec(key.getBytes(), "AES"),
        new IvParameterSpec(key.substring(0, 16).getBytes()));
    return aes.encryptHex(plainText);
}

// RSA加密
public String encryptByPublicKey(String plainText, String publicKey) {
    RSA rsa = new RSA(null, publicKey);
    return rsa.encryptBase64(plainText, KeyType.PublicKey);
}
```

### 4.2 数据脱敏

**脱敏规则**：

| 数据类型 | 脱敏规则 | 示例 |
|----------|----------|------|
| 手机号 | 中间4位* | 138****8000 |
| 身份证 | 中间10位* | 110***********1234 |
| 银行卡 | 中间保留前6后4 | 6222********1234 |
| 姓名 | 保留姓，名用* | 张** |
| 邮箱 | @前保留前3位，后用* | abc***@example.com |

**脱敏实现**：

```java
// 手机号脱敏
public String desensitizePhone(String phone) {
    if (StringUtils.isEmpty(phone) || phone.length() != 11) {
        return phone;
    }
    return phone.substring(0, 3) + "****" + phone.substring(7);
}

// 身份证脱敏
public String desensitizeIdCard(String idCard) {
    if (StringUtils.isEmpty(idCard) || idCard.length() != 18) {
        return idCard;
    }
    return idCard.substring(0, 3) + "***********" + idCard.substring(14);
}
```

| 增量备份 | 每小时 | 7天 |

## 六、应用安全

### 6.1 输入验证

**验证规则**：

| 输入类型 | 验证规则 |
|----------|----------|
| 字符串 | 长度、格式、特殊字符 |
| 数字 | 范围、类型 |
| 日期 | 格式、范围 |
| 文件 | 大小、类型、内容 |

**验证实现**：

```java
@Valid
public class UserDTO {
    @NotBlank(message = "用户名不能为空")
    @Length(min = 4, max = 20, message = "用户名长度4-20位")
    @Pattern(regexp = "^[a-zA-Z0-9_]+$", message = "用户名只能包含字母、数字、下划线")
    private String username;

    @NotBlank(message = "密码不能为空")
    @Length(min = 8, max = 20, message = "密码长度8-20位")
    @Pattern(regexp = "^(?=.*[a-z])(?=.*[A-Z])(?=.*\\d).+$", message = "密码必须包含大小写字母和数字")
    private String password;

    @NotBlank(message = "手机号不能为空")
    @Pattern(regexp = "^1[3-9]\\d{9}$", message = "手机号格式不正确")
    private String phone;
}
```

### 6.2 SQL注入防护

**防护措施**：

| 措施 | 实现 |
|------|------|
| 参数化查询 | 使用PreparedStatement |
| ORM框架 | 使用MyBatis-Plus |
| 输入过滤 | 过滤特殊字符 |
| 权限最小化 | 数据库账号最小权限 |

**安全实现**：

```java
// 安全：使用参数化查询
@Select("SELECT * FROM user WHERE username = #{username}")
User selectByUsername(@Param("username") String username);

// 不安全：字符串拼接（禁止使用）
@Select("SELECT * FROM user WHERE username = '" + username + "'")
User selectByUsername(String username);
```

### 6.3 XSS防护

**防护措施**：

| 措施 | 说明 |
|------|------|
| 输入过滤 | 过滤危险字符 |
| 输出编码 | HTML实体编码 |
| CSP策略 | 内容安全策略 |
| Cookie安全 | HttpOnly、Secure |

**输出编码**：

```java
// HTML编码
public String htmlEncode(String input) {
    return StringEscapeUtils.escapeHtml4(input);
}

// JavaScript编码
public String jsEncode(String input) {
    return StringEscapeUtils.escapeEcmaScript(input);
}
```

### 6.4 CSRF防护

**防护措施**：

| 措施 | 说明 |
|------|------|
| CSRF Token | 每次请求携带Token |
| SameSite Cookie | Cookie SameSite属性 |
| Referer验证 | 验证请求来源 |

**Token实现**：

```java
// 生成CSRF Token
public String generateCsrfToken() {
    return UUID.randomUUID().toString();
}

// 验证CSRF Token
public boolean validateCsrfToken(String token, String sessionToken) {
    return StringUtils.equals(token, sessionToken);
}
```

### 7.2 安全监控

**监控指标**：

| 监控项 | 阈值 | 告警级别 |
|--------|------|----------|
| 登录失败次数 | >5次/分钟 | 警告 |
| 异常请求 | >100次/分钟 | 严重 |
| SQL注入检测 | >1次 | 严重 |
| XSS检测 | >1次 | 严重 |
