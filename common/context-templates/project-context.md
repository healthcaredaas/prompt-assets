## 约束条件

### 技术约束
1. 遵循团队技术规范
2. 使用统一的代码风格
3. 遵循API设计规范
4. 满足安全合规要求

### 业务约束
1. 数据安全等级要求
2. 业务合规性要求
3. 性能指标要求
4. 可用性要求

### 质量约束
| 指标 | 目标值 |
|------|--------|
| 系统可用性 | 99.9% |
| 接口响应时间 | <200ms (P99) |
| 代码覆盖率 | >80% |
| 严重缺陷密度 | <0.5/KLOC |

1. 复制本模板到项目目录
2. 填充实际项目信息
3. 在 Prompt 中引用上下文：
   ```

   ```
4. 根据项目进展持续更新
